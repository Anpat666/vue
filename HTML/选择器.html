标签选择器（Type Selector）：使用 HTML 标签名称作为选择器，选取匹配的所有该标签的元素。 p { color: blue; } 这会将所有

元素的文本颜色设置为蓝色。 -------------------------------------------------------------------------------------- 通用选择器（Universal Selector）：使用 * 作为选择器，选取所有元素。 * { margin: 0; padding: 0; } 这会将所有元素的外边距和内边距都设置为零。 -------------------------------------------------------------------------------------- 类选择器（Class Selector）：使用类名作为选择器，选取具有相同类名的元素。 .button { background-color: #f0f0f0; border: 1px solid #ccc; } 这会将所有具有 class="button" 的元素设置相同的背景颜色和边框。 -------------------------------------------------------------------------------------- ID 选择器（ID Selector）：使用元素的唯一 ID 作为选择器。 #header { font-size: 24px; } 这会将具有 id="header" 的元素的字体大小设置为 24 像素。 -------------------------------------------------------------------------------------- 属性选择器（Attribute Selector）：选择具有特定属性或属性值的元素。 input[type="text"] { border: 1px solid #999; } 这会将所有类型为 "text" 的输入元素的边框设置为灰色。 -------------------------------------------------------------------------------------- 伪类选择器（Pseudo-class Selector）：选择元素的特定状态或位置。 a:hover { color: red; }
